# DIO_Linux_Script_DGU
Curso Linux Experience DIO

Pequeno scripts para criação e remoção de usuários, grupos e diretórios!

- Cria/remove usuários
- Cria/remove grupos
- Cria/remove diretórios
- Registra as alterações em LOG
